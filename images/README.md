# Carpeta de Imágenes

Coloca aquí las imágenes del proyecto Yoga, como fotos de clases, logos, etc.

Por ejemplo:
- yoga-placeholder.jpg: Imagen de placeholder para la sección de inicio.
- clase1.jpg: Foto de una clase de yoga.
- logo.png: Logo del sitio.

Recuerda optimizar las imágenes para web (formato JPG/PNG, tamaño reducido).
